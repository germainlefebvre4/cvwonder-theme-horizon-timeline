# CV Wonder Themes - Horizon Timeline

## Getting Started

```bash
cvwonder themes install https://github.com/germainlefebvre4/cvwonder-theme-horizon-timeline
cvwonder generate --input=cv.yml --theme=horizon-timeline
```
